# 光绘文件生成方法
## 先设置钻孔参数
 - `Manufacture` - `NC` - `NC Parameters`,只修改如下内容,其余不更改
   - `Format`:设置为`5`和`5`,表示整数5位，小数5位
   - [x] `leading zero suppression` ,打勾:表示坐标去除前导0
   - [x] `enhanced Excellon format` ,打勾:表示使用增强格式
## 设置钻孔符号
 - `Manufacture` - `NC` - `Drill Customization` , 设置好钻孔符号,无铜孔使用`X`符号
## 生成钻孔表
 - 先从`Artwork`中只显示钻孔层
 - `Manufacture` - `NC` - `Drill Legend` , 直接点`OK`，放在板框外面,板框里面全是钻孔符号
 - 钻孔图层在 `Manufacturing` - `Nclegend-1-x`,`x`:表示PCB层数,我的理解:两层板为`2`,四层板为`4`
## 生成钻圆孔文件
 - `Manufacture` - `NC` - `Drill` ,只修改下面内容
   - [x] `Auto tool select` ,打勾:表示自动选择钻孔工具
   - [x] `Optimize drill head travel` ,打勾:表示优化钻孔路径
   - 点击`Drill`
## 生成钻槽孔文件
 - `Manufacture` - `NC` - `Route` 
   - 点击`Route`
## 生成层叠顺序文件
 - `Manufacture` - `Cross Section Chat...`,只关系下面内容,其余不动
   - `Text block name`要设置好字体大小
   - `Chat Options`
     - [x] `Drill span`
     - [x] `Stacked vias`
     - [ ] `Backdrill span`
     - [ ] Embedded component legend`
   - `Display options` 
     - [x] `Drill label`
     - [x] `Layer name`
     - [x] `Layer type`
     - [x] `Layer material name`
     - [x] `Individual layer thickness`
     - [ ] `Thickness tolerance`
     - [ ] `Embedded status`
     - [ ] `Embedded attach method`
  - 点击`OK` ,放在板框外面合适位置不要挡住钻孔表
  - 层叠顺序图层在 `Manufacturing` - `Xsection_Chart`
## 生成光绘文件
 - `Manufacture` - `artwork`
   - 选择正确设置自己所有的光辉图层,选择要出的的图层
   - `Cteate Artwork`
## 打包光绘文件
 - 所有后缀: `.art`的文件 , 这是光绘文件
 - 所有后缀: `*.drl` 和 `*.rou` , 这是圆孔和槽孔文件
 - `art_param.txt` , 这是光绘参数文件
 - `nc_param.txt` , 这是钻孔参数文件
 - `nc_tools_auto.txt` , 这是钻孔工具文件
